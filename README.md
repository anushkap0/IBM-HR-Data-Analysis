# HR Attrition Analysis Project

## Overview
This project is an end-to-end HR analytics project built using Python, SQL, and Power BI.  
It analyzes employee attrition patterns to help understand which factors are associated with employees staying, terminating, or voluntarily resigning from the company.

The dataset used in this project contains employee-level information such as age, department, income, job involvement, work-life balance, and attrition status.

## Attrition Categories
The attrition column in this project is kept exactly as:
- Current employee
- Termination
- Voluntary Resignation

These values are used throughout the Python analysis, SQL queries, and dashboard visuals.

## Objectives
- Analyze employee attrition trends.
- Identify factors linked to employee turnover.
- Study the impact of department, gender, salary, job involvement, and work-life balance.
- Build clear visuals for business understanding.
- Create a portfolio-ready HR analytics project.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL
- Power BI

## Project Structure
- `Python file` – data cleaning, EDA, and visualizations.
- `SQL file` – queries for attrition analysis.
- `Dashboard` – Power BI report for insights and KPIs.
- `README.md` – project documentation.

## Python Analysis
The Python file performs:
- data loading,
- data type checking,
- missing value handling,
- duplicate checking,
- summary statistics,
- attrition-based grouping,
- visualization of key HR factors.

## SQL Analysis
The SQL file includes queries for:
- total employees by attrition type,
- overall attrition rate,
- attrition by department,
- attrition by gender,
- attrition by job role,
- attrition by education field,
- average income by attrition,
- overtime impact on attrition.

## Dashboard Insights
The Power BI dashboard shows:
- attrition rate,
- average age,
- average monthly income,
- employees left,
- attrition by department,
- attrition by gender,
- attrition by education field,
- job involvement and work-life balance patterns.

## Key Insights
- Attrition varies across departments and job roles.
- Lower income groups may show higher turnover.
- Job involvement and work-life balance are important workforce indicators.
- Current employees should be separated from termination and voluntary resignation for accurate analysis.

## Conclusion
This project provides a complete view of employee attrition using Python, SQL, and Power BI.  
It can be used as a portfolio project to demonstrate data cleaning, analysis, visualization, and business reporting skills.

## How to Run
1. Open the Python file and run the EDA code.
2. Run the SQL queries in your database tool.
3. Open the Power BI dashboard file.
4. Review the visuals and insights.
